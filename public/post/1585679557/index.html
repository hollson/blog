前言

  • Harbor是由VMware公司开源的企业级的Docker Registry管理项目，它包括权限管理(RBAC)、LDAP、日志审核、管理界面、镜像复制等功能。

安装Harbor

Harbor提供了离线安装和在线安装(推荐)两种方式，具体安装过程可参考官方安装向导。

1. Harbor的安装需要Docker和Docker-compose的支持，这里忽略该两项的安装过程。
2. 下载 Harbor离线包 ，安装到/usr/local/harbor目录下。

1
2
3
4

$ wget https://storage.googleapis.com/harbor-releases/release-1.9.0/harbor-offline-installer-v1.9.1.tgz
$ tar -zxvf harbor-offline-installer-v1.9.1.tgz -C /usr/local/
$ cd /usr/local/harbor && ls
# harbor.yml install.sh LICENSE prepare #包含的文件

3. Harbor推荐使用HTTPS方式访问， 官方也给出了详尽的Http配置向导。

# 生成注册表主机证书
$ cat > v3.ext <<-EOF
authorityKeyIdentifier=keyid,issuer
basicConstraints=CA:FALSE
keyUsage = digitalSignature, nonRepudiation, keyEncipherment, dataEncipherment
extendedKeyUsage = serverAuth
subjectAltName = @alt_names

[alt_names]
DNS.1=${HARBOR_HOST}
DNS.2=reg.mafool
EOF

  -out ${HARBOR_HOST}.crt

# https配置
https:
  port: 443
  # 证书文件(绝对路径)
  certificate: /usr/local/harbor/ssl/${HARBOR_HOST}.crt
  private_key: /usr/local/harbor/ssl/${HARBOR_HOST}.key

客户端访问

1. 如果Harbor宿主机位于局域网，则需要配置客户端HOSTS。

1
2

$ vim /etc/hosts    
192.168.1.x ${HARBOR_HOST}

2. 访问：https://${HARBOR_HOST}，如果有风险提示，忽略风险。登录系统后，修改密码并关闭对外注册功能。

**3. 终端登录（Mac版）：**进入Mac钥匙串/系统/证书，将reg.xxx.com证书拖进去并设置为 始终信任（证书可从服务端拷贝或从浏览器导出）,安装成功后记得`重启docker服务。

1
2

# 证书安装成功后，便可在终端登录harbor仓库
$ docker login ${HARBOR_HOST}
