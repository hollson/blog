安装JRE/JDK

官网 JRE： http://www.oracle.com/technetwork/java/javase/downloads/server-jre8-downloads-2133154.html
网盘 JDK： http://pan.baidu.com/s/1i4LeazV
网盘 JRE： http://pan.baidu.com/s/1i5p7NfR

  • 需要点击Accept License Agreement ，获取临时授权下载地址
  • 可选择通版本或服务器版的jre，JDK安装同理

$ wget http://download.oracle.com/otn-pub/java/jdk/8u131-b11/xxx/server-jre-8u131-linux-x64.tar.gz  
$ tar -zxvf server-jre-8u131-linux-x64.tar.gz -C /usr/local

$ vim /etc/profile            
#*************************修改环境变量***********************************
#在export PATH USER LOGNAME MAIL HOSTNAME HISTSIZE HISTCONTROL下添加
export JAVA_HOME=/usr/local/jdk1.8.0_131
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
#**********************************************************************

$ source /etc/profile          #刷新配置
$ java -version                #查看版本

安装Apache-Tomcat

1. 安装： 下载：http://tomcat.apache.org/download-80.cgi

1
2
3
4

java -version                                         #依赖于java
wget http://mirrors.shu.edu.cn/apache/tomcat/tomcat-9/v9.0.11/bin/apache-tomcat-9.0.11.tar.gz    #下载
tar -zxvf apache-tomcat-9.0.11.tar.gz -C /opt/        #解压
/opt/apache-tomcat-9.0.11/bin/startup.sh              #启动

2. 查看： http://x.x.x.x:8080/
3. 附： 若8080端口未开启，则进行如下操作：

1
2
3
4
5
6

yum install firewalld systemd -y                                #安装防火墙工具
firewall-cmd --state                                            #查看防火墙状态

firewall-cmd --zone=public --query-port=8080/tcp                #查看8080端口状态
firewall-cmd --zone=public --add-port=8080/tcp --permanent      #打开8080端口
firewall-cmd --reload

安装Apache

  • 官网：http://www.apache.org/
1
2
3

yum install -y httpd    #安装
systemctl start httpd    #启动
systemctl enable httpd

  • 常用操作
1
2
3
4
5

$ vim /etc/httpd/conf/httpd.conf

$ service httpd start         #启动
$ service httpd restart       #重新启动
$ service httpd stop          #停止服务
